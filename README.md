# garden-forge

Open-source projects, Codex skills, and practical AI tooling.

## Structure

- `projects/` - open-source project workspaces and reusable project assets.
- `skills/` - Codex skills and agent workflow packages.
- `templates/` - reusable scaffolds, prompts, configs, and starter layouts.
- `docs/` - documentation, notes, and public writeups.

## Current Focus

This repository is intended to collect practical AI tooling and reusable agent workflows that can be shared publicly.
