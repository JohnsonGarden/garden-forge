# Novel Harness File Map

## Me2AI

- `SPEC/Me2AI/人类作者创作初衷.md`: author intent, values, non-negotiables.
- `SPEC/Me2AI/outline.md`: macro structure, length target, act plan.
- `SPEC/Me2AI/写作风格.md`: style, prose, dialogue, pacing.
- `SPEC/Me2AI/故事整体设定.md`: world rules and setting boundaries.
- `SPEC/Me2AI/核心角色.md`: core cast, arcs, relationship priorities.

- `多智能体协作机制.md`: agent team and chapter workflow.

- `场景卡工作流.md`: scene-level planning rules.
- `章节审查工作流.md`: multi-agent review procedure.
- `章节评分表.md`: scoring dimensions and thresholds.

- `临时专家智能体机制.md`: temporary specialist rules.
- `会议纪要/`: per-chapter meeting minutes directory.

- `会议纪要/第X章章前启动会模板.md`: kickoff meeting template.
